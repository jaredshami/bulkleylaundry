# How to Add Your Images

## Quick Start Steps

### 1. Download Free Images

**Option A: Use Unsplash (Recommended for laundry business)**
- Go to https://unsplash.com
- Search: "laundry services", "clean clothes", "professional cleaning", "delivery service"
- Download high-resolution images
- Save to appropriate folders:
  - Hero images → `/images/hero/`
  - Service images → `/images/services/`
  - About/team images → `/images/about/`

**Option B: Use Canva (Create custom graphics)**
- Go to https://canva.com
- Create designs for service cards
- Export as JPG
- Save to appropriate folders

**Option C: AI Generate Images**
- Midjourney, DALL-E, or Stable Diffusion
- Prompts to use:
  - "Professional laundry service, clean clothes on hanger, modern laundry facility"
  - "Delivery person holding laundry bag, smiling customer"
  - "Professional dry cleaning rack with pressed garments"
  - "Team of laundry professionals at modern facility"

### 2. Optimize Your Images

Use these free tools to compress images:
- **TinyPNG**: https://tinypng.com (drag & drop)
- **Squoosh**: https://squoosh.app (Google's tool)
- **ImageOptim**: For Mac users

### 3. File Organization

```

├── hero/

├── about/
│   ├── location.jpg
│   ├── team.jpg
│   └── equipment.jpg
└── README.md
```

### 4. Upload Images to Your Site

1. Resize images to recommended dimensions (see README.md)
2. Compress using TinyPNG or Squoosh
3. Upload to the appropriate folder
4. No code changes needed - images will display automatically!

## Recommended Image Specifications

| Location | Dimensions | Max Size | Format |
|----------|-----------|----------|--------|
| Hero images | 1920x600px | 250KB | JPG |
| Service cards | 600x400px | 120KB | JPG |
| About section | 800x600px | 180KB | JPG |
| Background overlay | 1920x1080px | 300KB | JPG |
