## Image Recommendations

### Best Sources:
1. **Unsplash** (unsplash.com) - Free high-quality laundry/business images
2. **Pexels** (pexels.com) - Free stock photos
3. **Pixabay** (pixabay.com) - Free images and vectors
4. **Canva** (canva.com) - Create custom graphics and icons
5. **AI Generators** - Midjourney, DALL-E for custom service images

## Image Optimization Tips

### Format:
- **Hero images**: JPG (best for photos)
- **Icons**: SVG (scalable, crisp)
- **Service images**: JPG or WebP
- **Testimonials**: JPG (optimize for web)

### Size Guidelines:
- **Hero**: 1920x600px max (compress to 200-300KB)
- **Service cards**: 600x400px (80-150KB)
- **About images**: 800x600px (100-200KB)
- **Testimonials**: 200x200px (20-40KB)
- **Icons**: SVG (vector, any size)

### Optimization Tools:
- TinyPNG (tinypng.com)
- ImageOptim (imageoptim.com)
- Squoosh (squoosh.app)

## SEO Alt Text Guidelines

All images should have descriptive alt text for:
- Accessibility (screen readers)
- SEO (image search rankings)
- User experience (if image fails to load)

Example alt texts:
- "Professional laundry workers folding clean clothes"
- "Dry cleaning rack with pressed garments"
- "Delivery driver with customer's laundry order"
- "Bulkley Laundry team in front of storefront"
